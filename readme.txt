== Description ==

This plugin fixes the wordpress@domain.com emails and sets it to the Admin email found under General > Settings.

== Installation ==

1. Upload `admin-email-fix.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Where is the settings page =

There is no settings/options page. The plugin takes the admin email from General > Settings.
